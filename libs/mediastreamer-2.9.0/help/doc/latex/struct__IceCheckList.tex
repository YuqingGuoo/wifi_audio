\subsection{Detailed Description}
Structure representing an ICE check list.

Each media stream must be assigned a check list. Check lists are added to an ICE session using the \doxyref{ice\_\-session\_\-add\_\-check\_\-list()}{p.}{ice_8h_ae3899b071d5d6b981ff85c7d42f91077} function.
